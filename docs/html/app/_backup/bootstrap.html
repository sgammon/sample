bootstrap Module¶

This small utility file is loaded by AppTools and executed as early as possible in the execution flow, to facilitate the injection of paths onto sys.path, preloading, and other process-level init for the app.

It’s dead simple to use:

# -*- coding: utf-8 -*-

import bootstrap
bootstrap.AppBootstrapper.prepareImports().preload()

class bootstrap.AppBootstrapper[source]¶

Bases: object

Performs early app bootstrap procedures, such as adding to sys.path and preloading apptools.

Paths from cls.injected_paths are added to sys.path if they are not present. This class is designed to be idempotent - a module-global flag (_bootstrapped) is flipped once the bootstrapper has run.
